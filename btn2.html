How to declare an array?

To declare an array in java script you need to create a variable - with the command 'var' - and state the name you want your array to be followed by an equals sign :

eg.(if you want an array for the name of some fruits)

var fruits =

Then with square brackets list your different items you want in your array in quotation marks. Separate the items with a comma and a space.

eg.(following the above example)

var fruits = ["orange", "apple", "banana"]
